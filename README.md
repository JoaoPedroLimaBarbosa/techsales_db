# TechSales DB — Controle de Acesso no PostgreSQL

## Objetivo
Configurar usuários, roles e permissões de acesso ao banco `techsales_db` conforme políticas da empresa.

## Perfis de Acesso
- **Vendedor** → Inserção de vendas
- **Analista** → Consulta de relatórios
- **Admin_db** → Administração total e controle de permissões

## Estrutura
Scripts SQL organizados por etapas (criação, permissões e testes) e evidências dos testes.

## Testes
Inclui prints comprovando o funcionamento e erros esperados de cada perfil.
